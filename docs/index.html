Trentino Territory & Tourism Facilties

Dongili Lorenzo, Grotto Giulia, Sacchet Stefano

View Project

Purpose

This project discusses the development of a Knowledge Graph which aim is to provide an easy and convenient access about key tourism facilities in the region, such as hotels, restaurants, museums, natural parks, and other attractions. By integrating various knowledge and data resources, this KG will serve as a foundational tool for applications and services focused on delivering comprehensive and user-friendly information about the Trentino region.
